# PulseNetwork

PulseNetwork is a real-time responsive web application to find nearest blood donors, manage blood inventories, predict demand using smart ML analytics, and chat securely between donors and blood banks.

## Features
- **Accurate Division Map of Bangladesh**: Real-time interactive SVG map of Bangladesh featuring 8 divisions (including Mymensingh) with precise geographic boundaries.
- **AI/ML Demand Prediction**: Predict blood demand dynamically based on history.
- **Real-Time Live Chat**: Instant Messaging support via WebSockets.
- **Interactive UI**: Futuristic neon layout with glowing visual markers and particles.

## Getting Started
To run the server locally:
```bash
npm install
npm start
```
Go to `http://localhost:3000` to access the application.
