# PulseNetwork 🩸
**রিয়েল-টাইম রক্তদান এবং জরুরি যোগাযোগ নেটওয়ার্ক**

PulseNetwork হলো বাংলাদেশ জুড়ে রক্তদাতা, ব্লাড ব্যাংক এবং জরুরি রক্তের প্রয়োজনে থাকা রোগীদের মধ্যে সরাসরি সংযোগ তৈরি করার জন্য একটি সহজ এবং প্রিমিয়াম প্ল্যাটফর্ম। এটি সম্পূর্ণ রিয়েল-টাইম লাইভ ডাটার ওপর ভিত্তি করে কাজ করে।

## 📊 প্রজেক্ট প্রেজেন্টেশন (Presentation Slide Details)

আমরা আমাদের প্রেজেন্টেশনে যে তথ্যগুলো ব্যবহার করেছি, তা নিচে দেওয়া হলো যাতে যে কেউ সহজেই পড়ে প্রজেক্টের গুরুত্ব ও কাজ বুঝতে পারেন:

### ১. কেন এই প্রজেক্টটি দরকার? (The Problem)
যখন আমাদের পরিবার বা পরিচিত কারও হঠাৎ রক্তের প্রয়োজন হয়, আমরা সাধারণত ফেসবুকে পোস্ট করি অথবা পরিচিতদের ফোন করি। এতে অনেক সময় নষ্ট হয় এবং জরুরি রোগীকে সময়মতো রক্ত দেওয়া সম্ভব হয় না। 
- **ভুল ও পুরোনো নাম্বার:** ফেসবুক বা বিভিন্ন গ্রুপে থাকা রক্তদাতাদের ফোন নাম্বার অনেক সময় বন্ধ বা পুরোনো থাকে।
- **দূরত্ব ও ম্যাপের অভাব:** রোগী ও ডোনারের দূরত্ব কত তা ম্যাপে না দেখে সহজে অনুমান করা যায় না।
- **ব্লাড ব্যাংকের অসচ্ছতা:** কোন হাসপাতালে কোন গ্রুপের রক্ত রেডি আছে তা তাৎক্ষণিক জানার কোনো উপায় ছিল না।

### ২. আমাদের সমাধান (The Solution)
PulseNetwork এই মূল্যবান সময় বাঁচাতে সাহায্য করে। এটি একটি ডিজিটাল মানচিত্রের মতো, যেখানে আপনি এক ক্লিকেই দেখতে পাবেন আপনার সবচেয়ে কাছাকাছি এই মুহূর্তে কোন রক্তদাতার কী গ্রুপ আছে এবং সরাসরি তার সাথে মেসেজে কথা বলতে পারবেন।
- **ইন্টারেক্টিভ বাংলাদেশ ম্যাপ:** ম্যাপে ক্লিক করে যেকোনো বিভাগের লাইভ রক্তদাতার তথ্য মুহূর্তেই জানা যায়।
- **রিয়েল-টাইম লাইভ চ্যাট:** ডোনারের সাথে সরাসরি যোগাযোগের জন্য মেসেঞ্জারের মতো ইনস্ট্যান্ট চ্যাট সিস্টেম।
- **স্মার্ট ইনভেন্টরি ট্র্যাকিং:** ব্লাড ব্যাংকগুলোর লাইভ রক্তের স্টক দেখার সুবিধা।

### ৩. প্রজেক্টের বর্তমান অবস্থা (Current Status)
প্রজেক্টটি বর্তমানে সম্পূর্ণ প্রস্তুত এবং রেন্ডার ক্লাউড সার্ভারে লাইভ চলছে। এর মূল ফিচারগুলো হলো:
- **৮টি বিভাগের নিখুঁত মানচিত্র:** ঢাকা ও ময়মনসিংহ আলাদা বিভাগ হিসেবে নিখুঁত সীমানা সহ glowing নিওন থিমে ম্যাপে দৃশ্যমান।
- **লাইভ চ্যাটিং:** রোগী ও ডোনারের মধ্যে তাৎক্ষণিক মেসেজ আদান-প্রদান করা যায়।
- **ডোনার কুলডাউন পলিসি:** কোনো ডোনার একবার রক্ত দিলে তার আইডি স্বয়ংক্রিয়ভাবে ৯০ দিনের জন্য হাইড হয়ে থাকে, যাতে তার স্বাস্থ্যের ক্ষতি না হয়।
- **১-ক্লিক ডেমো ড্যাশবোর্ড:** এডমিন, ডোনার, ও ব্লাড ব্যাংক ভিউ সরাসরি পরীক্ষা করার সুবিধা।

### ৪. ভবিষ্যৎ পরিকল্পনা (Future Roadmap)
সেবাকে আরও উন্নত ও দেশব্যাপী ছড়িয়ে দিতে আমাদের পরিকল্পনাগুলো:
- **অফলাইন এসএমএস এলার্ট (Offline SMS):** ইন্টারনেট না থাকলেও সাধারণ মোবাইল মেসেজের মাধ্যমে নিকটস্থ ডোনারদের ফোনে সরাসরি রিকোয়েস্ট পাঠানো ও রক্ত খোঁজার সুবিধা।
- **মোবাইল অ্যাপ্লিকেশন:** অ্যান্ড্রয়েড এবং আইওএস ফোনের জন্য ডেডিকেটেড মোবাইল অ্যাপ তৈরি।
- **ভেরিফাইড ডোনার ব্যাজ:** জাতীয় পরিচয়পত্র (NID) ভেরিফিকেশন এবং সক্রিয় ডোনারদের জন্য সম্মাননা ব্যাজ।
- **হাসপাতাল নেটওয়ার্ক সংযোগ:** দেশের সকল বড় হাসপাতালের ব্লাড ব্যাংক ইন্টিগ্রেশন।

### ৫. ব্যবহৃত প্রযুক্তি (Technical Stack)
- **ফ্রন্টএন্ড:** Next.js (React Framework) & Tailwind CSS (নিওন ও রেসপন্সিভ ডিজাইনের জন্য)
- **ব্যাকএন্ড:** Node.js & Express.js (এপিআই ও ব্যাকএন্ড লজিক)
- **রিয়েল-টাইম চ্যাট:** WebSockets (ws লাইব্রেরি)
- **ডাটা সংরক্ষণ:** JSON ফাইল ডাটাবেজ ও Render persistent disk মাউন্ট।

## 🛠️ রান করার নিয়ম (Installation Guide)

### ১. লোকাল কম্পিউটারে রান করার নিয়ম:
প্রজেক্টটি ডাউনলোড করার পর টার্মিনালে নিচের কমান্ডগুলো রান করুন:

```bash
# ডিপেন্ডেন্সি ইনস্টল করুন
npm install

# Next.js ফ্রন্টএন্ড কোড বিল্ড করুন (এটি ফাইলগুলো Express-এর public ফোল্ডারে পাঠিয়ে দেবে)
npm run build

# ব্যাকএন্ড সার্ভার চালু করুন
npm start
```
ব্রাউজারে গিয়ে ওপেন করুন: `http://localhost:3000`

### ২. সার্ভারে ডেপ্লয় করার নিয়ম (Render Deploy):
প্রজেক্টের রুটে একটি `render.yaml` ফাইল দেওয়া আছে। রেন্ডারের **Blueprints** অপশনে গিয়ে এই গিটহাব রিপোজিটরিটি কানেক্ট করলেই রেন্ডার নিজে থেকে বিল্ড ও ডেপ্লয় করে নেবে।
