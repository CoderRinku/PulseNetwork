PulseNetwork

রিয়েল-টাইম রক্তদান এবং ব্লাড ব্যাংক নেটওয়ার্কিং প্ল্যাটফর্ম

একটি মডার্ন ফুল-স্ট্যাক ওয়েব অ্যাপ্লিকেশন যা বাংলাদেশ জুড়ে রক্তদাতা, ব্লাড ব্যাংক এবং রোগীদের মধ্যে ইনস্ট্যান্ট ডেটা শেয়ারিং ও লাইভ চ্যাট সুবিধা নিশ্চিত করে।

প্রজেক্টটির প্রয়োজনীয়তা (Problem Statement vs Solution)

🚨 বর্তমান সমস্যাসমূহ (The Challenges)

  • • রক্ত খোঁজার চিরাচরিত পদ্ধতিগুলো (যেমন: ফেসবুক পোস্ট, ডিরেক্টরি) ধীরগতির এবং অনেক সময় তথ্য ভুল থাকে।
  • • রোগীরা মানচিত্রে ভৌগোলিকভাবে নিকটতম ডোনারদের অবস্থান দেখতে পারে না।
  • • হাসপাতাল ও ব্লাড ব্যাংকগুলোর কাছে রক্তের স্টক কত ব্যাগ আছে তা লাইভ ট্র্যাক করার উপায় নেই।

✅ PulseNetwork-এর সমাধান (The Solution)

একটি সমন্বিত লাইভ ইকোসিস্টেম তৈরি করা, যেখানে ভৌগোলিক মানচিত্র এবং রিয়েল-টাইম যোগাযোগের মাধ্যমে তাত্ক্ষণিকভাবে রক্তের উৎস খুঁজে বের করা যায়।

  • ✔ নিখুঁত ৮টি প্রশাসনিক বিভাগের ইন্টারেক্টিভ SVG ম্যাপ।
  • ✔ রোগীদের সাথে ডোনারদের সংযোগের জন্য লাইভ WebSocket চ্যাট।

সিস্টেম আর্কিটেকচার ও ডেটা ফ্লো (Architecture & Data Flow)

🖥️

Frontend Layer (React/Next.js)

ইউজার ইন্টারফেস, মানচিত্র এবং লাইভ চ্যাট উইন্ডো রেন্ডার করে। REST API এবং WebSocket ক্লায়েন্ট ধারণ করে।

→
⚙️

Backend Server (Node.js/Express)

সার্ভার প্রসেস, যা CORS পলিসি মেনে রিকোয়েস্ট নেয়, WebSocket কানেকশন নিয়ন্ত্রণ করে এবং ML অ্যালগরিদম রান করে।

💾

Data Layer (JSON Database)

ফাইল-বেসড ডাটাবেজ (`database.json`), যা রেন্ডার ক্লাউডের Persistent Disk-এর মাধ্যমে ডেটা স্থায়ীভাবে ধরে রাখে।

📊 কীভাবে ডেটা প্রবাহিত হয়? (How Data Flows)

  1. ইউজার ম্যাপে ক্লিক করলে বা ডোনার সার্চ করলে ব্রাউজার থেকে Express API-তে HTTP রিকোয়েস্ট যায়।
  2. Express সার্ভার `database.json` ফাইল রিড করে ইনস্ট্যান্ট রেসপন্স ফেরত পাঠায়।
  3. রোগী ডোনারকে মেসেজ পাঠালে তা **WebSocket Protocol** ব্যবহার করে সার্ভার সরাসরি কানেক্টেড ডোনারের স্ক্রিনে পৌঁছে দেয়।

ডেটাবেজ ডিজাইন ও স্কিমা (Database Schema & Entities)

1. Users Table (ব্যবহারকারী টেবিল)

Field (ফিল্ড) Type (টাইপ) Description (বিবরণ)

Field (ফিল্ড) Type (টাইপ) Description (বিবরণ)

is_eligibleBooleanরক্তদানের যোগ্যতা (৯০ দিনের কুলডাউন)

প্রধান ফিচারের কারিগরি দিক (Core Features & Implementation)

🗺️

ইন্টারেক্টিভ জিও-ম্যাপ

বাংলাদেশের GeoJSON বাউন্ডারি কনভার্ট করে তৈরি কাস্টম SVG ম্যাপ, যা রিঅ্যাক্ট স্টেট ও প্রজেক্টেড কোঅর্ডিনেট ব্যবহার করে লাইভ ডোনারের সংখ্যা ম্যাপের ওপর দেখায়।

💬

রিয়েল-টাইম চ্যাট (WebSockets)

Node-ws ব্যবহার করে তৈরি ডাবল-ওয়ে চ্যাট চ্যানেল, যা সেশন টোকেন অথেন্টিকেশনের মাধ্যমে রোগীদের এবং রক্তদাতাদের সরাসরি সংযুক্ত করে।

⏱️

৯০ দিনের কুলডাউন লুপ

রক্তদানের পর ডোনারের আইডি স্বয়ংক্রিয়ভাবে ব্যাকএন্ড ডেট ক্যালকুলেটর দ্বারা ৯০ দিনের জন্য সার্চ থেকে হাইড রাখা হয়।

📊

এমএল ডিমান্ড ফোরকাস্ট

ইনভেন্টরি ডেটা বিশ্লেষণ করে হাসপাতালের রক্ত সংকটের পূর্বাভাস গণনা করার জন্য ব্যাকএন্ডে একটি লিনিয়ার রিগ্রেশন এলগরিদম রান করে।
